Windows 10 环境部署与运行教学

1

安装 Python 运行环境

前往 Python 官网 下载最新版 (推荐 3.10 或更高)。

⚠️ 重要：

安装时务必勾选 "Add Python to PATH" 选项！否则无法在命令行运行。

2

解压代码压缩包

将从 GitHub 下载的 ZIP 文件右键解压到任意文件夹（例如桌面）。

3

安装依赖库

在解压后的文件夹空白处，按住 Shift 并右键，选择“在此处打开 PowerShell 窗口”或“命令提示符”。

输入以下命令并回车：

pip install -r requirements.txt

注：如果提示 pip 不是内部命令，说明步骤 1 的 PATH 没勾选，请重新安装 Python。

4

启动程序

在同一个窗口中输入以下命令启动：

python vibecontrol_pro.py

✨ 核心功能升级

全功能录制

支持鼠标左右中键、滚轮、拖拽以及键盘所有按键的录制与回放。

高精度时间

毫秒级精度，严格遵守录制时的操作间隔，完美复原操作流程。

增强型点击器

支持双击、三击、随机时间偏移（防检测）以及自定义按下时长。

任务管理

支持任务的导入、导出、删除以及自定义保存路径。

🚀 进阶：生成独立 EXE 可执行文件

如果你想把程序打包成一个双击即可运行的 .exe 文件，请执行：

1. 安装打包工具：pip install pyinstaller

2. 执行打包命令：

pyinstaller --noconsole --onefile vibecontrol_pro.py

3. 在生成的 dist 文件夹中找到 vibecontrol_pro.exe。

🔍 常见问题排查

  • 提示 ModuleNotFoundError: 说明依赖没装全，请重新执行步骤 3。
  • 程序启动后没反应: 检查是否有杀毒软件拦截了模拟点击操作。
  • 紧急停止: 运行中如需停止，请按键盘左上角的 ESC 键。
